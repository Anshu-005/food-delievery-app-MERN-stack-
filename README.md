**Food Delivery App (MERN Stack)**
A full-stack food delivery application built with the MERN stack (MongoDB, Express.js, React, Node.js).
This project demonstrates my ability to design scalable backend APIs, craft responsive frontend UIs, and integrate real-world features like authentication, payments, and order tracking.

  Why This Project Matters
- Showcases end-to-end development skills (frontend, backend, database).
- Highlights problem-solving in cart management, authentication, and order flow.
- Demonstrates UI/UX sensitivity with responsive design and dark/light mode.
- Reflects real-world product thinking: payments, delivery tracking, and user engagement.

  Tech Stack
- Frontend: React, Context API, Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Auth: JWT, bcrypt

  Key Points:
- Built with scalability and maintainability in mind.
- Focused on clean repo structure and defensive coding practices.
- Designed to be student-friendly and intuitive, aligning with my interest in supportive tech.
